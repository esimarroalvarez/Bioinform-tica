# Bioinformatica
Análisis de expresión géncia (RNA-seq). Proyecto de bioinformática para análisis de expresión usando un dataset de GEO (GSE266525).

Dataset
- raw_counts.csv: conteos de RNA-seq
- samples.csv: metadatos de las muestras

Objetivos:
- Normalizar los datos (log2)
- Calcular estadísticas básicas
- Visualizar resultados con boxplot y heatmap
